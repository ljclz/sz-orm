快捷查询（Db::name 风格）

对应 think-orm 的 Db::name('user')->where(...)->select() API。 无需定义 Model 即可直接基于表名查询/插入/更新/删除。

§与 QueryBuilder 的关系

QueryBuilder<M> 要求泛型参数 M: Model，适合已知 Model 类型的场景。 QuickQuery 则用 () 占位 Model，仅依赖表名，避免为临时查询定义 Model。

§用法

use sz_orm_core::quick_query::Db;
use sz_orm_core::{get_dialect, DbType, Value};

let dialect = get_dialect(DbType::MySQL).unwrap();
// SELECT * FROM users WHERE age > 18 ORDER BY id DESC LIMIT 10
let sql = Db::new(dialect).name("users")
    .where_cond("age > 18")
    .order_desc("id")
    .limit(10)
    .build_select();
