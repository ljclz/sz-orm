Struct ScopeRegistry

pub struct ScopeRegistry { /* private fields */ }
Expand description

全局作用域注册表

管理多个 GlobalScope 的启用/禁用状态。 典型用法：临时禁用软删除作用域以查询已删除行（without_scope）。
