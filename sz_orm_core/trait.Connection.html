Expand description

数据库连接 trait

注意：此 trait 手动解糖 async 方法（不使用 #[async_trait]）， 以避免 &str 参数触发 HRTB 与 sqlx::Executor 冲突。 所有 async 方法使用单一生命周期 'a（绑定 &'a mut self 和 &'a str）， 而非 HRTB，从而允许 sqlx 适配器实现。
