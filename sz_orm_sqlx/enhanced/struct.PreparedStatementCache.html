Struct PreparedStatementCache

pub struct PreparedStatementCache { /* private fields */ }
Expand description

预备语句缓存（LRU 策略）。

缓存 SQL 语句到预备语句 ID 的映射，避免重复解析和编译 SQL。 使用 LRU（Least Recently Used）策略在容量满时驱逐最久未使用的条目。

§线程安全

内部使用 Mutex 保护，可安全跨线程共享。 LRU 排序基于单调递增的原子计数器，不受系统时钟精度影响。
