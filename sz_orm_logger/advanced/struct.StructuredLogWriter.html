Struct StructuredLogWriter 

Source 
pub struct StructuredLogWriter { /* private fields */ }
Expand description

结构化日志器：支持结构化字段、target 过滤与多 sink 输出。

结合 LevelFilter 与 LogSink，提供生产级日志能力：

  1. 按 target 过滤日志级别
  2. 记录结构化字段
  3. 扇出到多个输出后端
