§QoS 2 四次握手状态机

实现 MQTT QoS 2（Exactly Once）消息传输的完整四次握手流程：

发送方                接收方
  |  PUBLISH(packet_id)  ->    |
  |  <-  PUBREC(packet_id)     |
  |  PUBREL(packet_id)   ->    |
  |  <-  PUBCOMP(packet_id)    |

发送方状态：Init -> Published -> Received -> Released -> Complete 接收方状态：Init -> Received -> Released -> Complete

任何阶段异常可通过 PacketState 查询并支持重传。
