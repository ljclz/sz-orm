pub struct Summary { /* private fields */ }
Expand description

Summary 指标，基于排序数组计算分位数。

与 Prometheus Summary 类似，记录所有观测值并按需计算分位数。 适合样本量可控的场景；超大规模样本应考虑 T-Digest 近似算法。

计算指定分位数的值（0.0..=1.0）

使用最近排名法（Nearest Rank）：rank = ceil(p * n)，至少为 1。 返回 None 表示无样本或分位数超出范围。
