钩子系统 — 软删除 + 全局作用域

演示 HookContext、HookRegistry、ScopeRegistry 的用法。 Hookable + SoftDelete + GlobalScope 让 Model 自动获得：

  • 查询时自动追加 deleted_at IS NULL
  • 删除时改为 UPDATE SET deleted_at = NOW()

运行：cargo run -p sz-orm-examples --bin hooks_soft_delete
