pub fn add_task(&mut self, task: DagTask) -> Result<(), SchedulerError>

添加任务节点

验证依赖的任务是否已存在，防止引用不存在的依赖。 若需在所有任务注册后再配置依赖（例如构建循环以测试环检测）， 请使用 Self::add_dependency。

Source

pub fn add_dependency( &mut self, task_id: &str, dep_id: &str, ) -> Result<(), SchedulerError>

为已存在的任务追加依赖关系。

与 Self::add_task 不同，此方法允许在所有任务注册后配置依赖， 因此可以构建出循环依赖以供 Self::has_cycle 检测。 若任务或依赖不存在则返回错误。
