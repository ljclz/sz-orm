OAuth2 授权码流程（Authorization Code Flow）

实现 RFC 6749 Section 4.1 的授权码流程：

  1. 客户端重定向用户到授权服务器
  2. 用户授权后，授权服务器返回授权码
  3. 客户端用授权码交换访问令牌

本模块提供流程状态管理，不包含 HTTP 传输层实现。
