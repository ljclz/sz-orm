Struct OAuth2Server

pub struct OAuth2Server { /* private fields */ }
Expand description

OAuth2 授权服务器：管理授权码的签发、验证与交换。

内部使用 Mutex<HashMap> 存储授权码，支持：

  • 创建授权码（create_authorization_code）
  • 用授权码交换令牌（exchange_code）
  • 验证客户端凭据
  • 授权码一次性消费
